# 🤖 Mizu-Intelligence: Enterprise RAG & Voice Orchestrator

Sistema de arquitectura desacoplada que integra reconocimiento de voz en tiempo real con un motor de IA basado en RAG (Retrieval-Augmented Generation). Diseñado para escalar el conocimiento de NPCs en entornos virtuales o asistentes corporativos.

## 🚀 Key Features
- **LLM Orchestration:** Flujo de razonamiento asíncrono con FastAPI y LangChain.
- **Advanced RAG:** Recuperación semántica utilizando ChromaDB y OpenAI Embeddings.
- **Hybrid Interface:** Frontend en JS para síntesis/reconocimiento de voz y Backend en Python para inferencia.
- **Production Ready:** Validación de datos con Pydantic y manejo de streaming de respuestas.

## 🛠️ Tech Stack
- **Backend:** Python 3.10+, FastAPI, LangChain.
- **IA:** OpenAI GPT-4o, ChromaDB (Vector Store).
- **Frontend:** JavaScript (Web Speech API), CSS3 (Tailwind-like).
